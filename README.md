## ✅ Overview
**Rabbit** is a modern, animated, and responsive **K6 Report Test Analyzer** frontend built with **Next.js** and powered by **Tailwind CSS**, **Framer Motion**, and custom UI components.  
It processes pasted **K6 load test output**, generates **interactive reports**, and provides **visual insights** for better performance analysis.  

---

## 🎯 Key Features

- ✅ **Modern UI** with smooth animations powered by *Framer Motion*
- ✅ **Responsive Design** using *Tailwind CSS*
- ✅ **Paste & Parse K6 Output** in real-time
- ✅ **Interactive Charts & Graphs**
- ✅ **Detailed Test Insights** with metrics, checks, and performance trends
- ✅ **Theme Support** (Light/Dark mode)
- ✅ **Badge Indicators** for Pass/Fail statuses
- ✅ **Fast Rendering** with *Next.js Optimizations*
